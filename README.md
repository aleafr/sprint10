# Sprint 10 — Estudo de Mercado: Restaurantes em Los Angeles

Projeto desenvolvido durante o bootcamp de Data Analytics da TripleTen.

## Objetivo

Realizar uma análise de mercado sobre restaurantes em Los Angeles para apoiar a abertura de uma cafeteria com garçons robôs.

## Descrição

Neste projeto foram realizadas:
- preparação e limpeza dos dados;
- análise exploratória de restaurantes em Los Angeles;
- estudo dos tipos de estabelecimentos;
- análise de cadeias e restaurantes independentes;
- análise da quantidade de assentos;
- criação de gráficos e visualizações;
- interpretação dos resultados;
- desenvolvimento de recomendações para o negócio.

## Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
